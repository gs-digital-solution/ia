Corrigé de l'exercice sur la fonction \(f(x)=\frac{x-1}{x+2}\)

1. Étude de la fonction \(f\) et construction de sa courbe représentative

a) Domaine de définition :

La fonction \(f\) est définie pour \(x\neq-2\).

Domaine : \(\mathcal{D}_f=\mathbb{R}\setminus\{-2\}\).

b) Limites et asymptotes :

Limite en \(-2\) :

\[\lim_{x\to-2^-}f(x)=-\infty,\quad\lim_{x\to-2^+}f(x)=+\infty\]

⇒ Asymptote verticale : \(x=-2\).

Limite en \(\pm\infty\) :

\[\lim_{x\to\pm\infty}f(x)=\lim_{x\to\pm\infty}\frac{x}{x}=1\]

⇒ Asymptote horizontale : \(y=1\).

c) Dérivée et variations :

\[f'(x)=\frac{(1)(x+2)-(x-1)(1)}{(x+2)^2}=\frac{3}{(x+2)^2}\]

\(f'(x)>0\) pour tout \(x\neq-2\) ⇒ \(f\) est strictement croissante sur \(]-\infty,-2[\) et \(]-2,+\infty[\).

d) Tableau de variations :

\(x\) \(-\infty\) \(-2\) \(+\infty\)
\(f'(x)\) + ND +
\(f(x)\) \(1\) ↗ \(-\infty\mid+\infty\) ↗ \(1\)

e) Courbe représentative :

- La courbe \((\epsilon)\) admet une asymptote verticale en \(x=-2\) et une asymptote horizontale en \(y=1\).

- Elle est croissante sur chaque intervalle de son domaine.

- Points remarquables : \(f(1)=0\), \(f(0)=-\frac{1}{2}\), \(f(-1)=-2\).

2. Résolution graphique et algébrique du système d'inéquations :

\[0<\frac{x-1}{x+2}<4\]

a) Résolution graphique :

On cherche les valeurs de \(x\) pour lesquelles la courbe \((\epsilon)\) est strictement entre les droites \(y=0\) et \(y=4\).

Graphiquement, on observe que cela correspond à :

\[x\in]-\infty,-3[\cup]1,+\infty[\]

b) Résolution par le calcul :

Première inéquation : \(\frac{x-1}{x+2}>0\)

Étude du signe :

\(x\) \(-\infty\) \(-2\) \(1\) \(+\infty\)
\(x-1\) - - 0 +
\(x+2\) - 0 + +
\(f(x)\) + ND 0 +

⇒ \(f(x)>0\) pour \(x\in]-\infty,-2[\cup]1,+\infty[\)

Deuxième inéquation : \(\frac{x-1}{x+2}<4\)

\[\frac{x-1}{x+2}-4<0\Rightarrow\frac{x-1-4(x+2)}{x+2}<0\Rightarrow\frac{-3x-9}{x+2}<0\]

Simplification :

\[\frac{-3(x+3)}{x+2}<0\Rightarrow\frac{x+3}{x+2}>0\]

Étude du signe :

\(x\) \(-\infty\) \(-3\) \(-2\) \(+\infty\)
\(x+3\) - 0 + +
\(x+2\) - - 0 +
\(\frac{x+3}{x+2}\) + 0 - +

⇒ \(\frac{x+3}{x+2}>0\) pour \(x\in]-\infty,-3[\cup]-2,+\infty[\)

Intersection des deux conditions :

- \(f(x)>0\) pour \(x\in]-\infty,-2[\cup]1,+\infty[\)

- \(f(x)<4\) pour \(x\in]-\infty,-3[\cup]-2,+\infty[\)

⇒ Solution : \(x\in]-\infty,-3[\cup]1,+\infty[\)

Solution finale :

\[0<\frac{x-1}{x+2}<4\quad\Leftrightarrow\quad x\in]-\infty,-3[\cup]1,+\infty[\]

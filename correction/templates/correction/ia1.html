Exercice 1 (3 points)

1. Divisibilité par 3 et 11

Soit \(N = 26x95y\) avec \(x,y \in \{0,1,...,9\}\)

Divisibilité par 3 : \(2+6+x+9+5+y = 22+x+y\) divisible par 3

⇒ \(x+y \equiv 2 \pmod{3}\)

Divisibilité par 11 : \((2+x+9+y) - (6+5) = (11+x+y) - 11 = x+y\) divisible par 11

⇒ \(x+y = 0\) ou \(x+y = 11\)

Système :

- \(x+y \equiv 2 \pmod{3}\)

- \(x+y = 0\) ou \(11\)

Si \(x+y = 0\) ⇒ \(x=y=0\) mais alors \(x+y=0 \not\equiv 2 \pmod{3}\) ❌

Si \(x+y = 11\) ⇒ \(11 \equiv 2 \pmod{3}\) ✓

Donc \(x+y = 11\) avec \(x,y\) chiffres.

Solutions : (2,9), (3,8), (4,7), (5,6), (6,5), (7,4), (8,3), (9,2)

2. Puissances de 7

a) Chiffre des dizaines de \(7^n\)

Calculons les premières puissances :

- \(7^1 = 7\) → dizaines : 0

- \(7^2 = 49\) → dizaines : 4

- \(7^3 = 343\) → dizaines : 4

- \(7^4 = 2401\) → dizaines : 0

- \(7^5 = 16807\) → dizaines : 0

- \(7^6 = 117649\) → dizaines : 4

On observe une périodicité modulo 4 :

- Si \(n \equiv 1 \pmod{4}\) : dizaines = 0

- Si \(n \equiv 2 \pmod{4}\) : dizaines = 4

- Si \(n \equiv 3 \pmod{4}\) : dizaines = 4

- Si \(n \equiv 0 \pmod{4}\) : dizaines = 0

b) Reste de \(7^{77}\) modulo 4

\(7 \equiv -1 \pmod{4}\) ⇒ \(7^{77} \equiv (-1)^{77} \equiv -1 \equiv 3 \pmod{4}\)

c) Reste de \(7^{777}\) modulo 100

On cherche le cycle modulo 100 :

- \(7^1 = 7\)

- \(7^2 = 49\)

- \(7^3 = 343 \equiv 43\)

- \(7^4 = 2401 \equiv 1\)

Cycle d'ordre 4.

\(777 = 4 \times 194 + 1\) ⇒ \(7^{777} \equiv 7^1 \equiv 7 \pmod{100}\)

Partie B (Compétences)

Problème militaire

Soit :

- n soldats par régiment

- k places par salle

- Équation : \(11n=7k+5\) ⇒ \(11n\equiv5\pmod{7}\)

\(11\equiv4\pmod{7}\) ⇒ équation : \(4n\equiv5\pmod{7}\)

Inverse de 4 mod7 est 2 car \(4\times2=8\equiv1\) ⇒ multiplions par l'inverse :

\(n\equiv10\equiv3\pmod{7}\)

Donc n=7q+3 avec n<300.

Maximal ⇒ q maximal tel que n<300 ⇒ q=42 ⇒ n=297
